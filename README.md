# 概要
这是一个基于WEB-RTC和Java WebSocket的一个浏览器时时在线视频通话的软件。使用了ssh的Java架构。前后端分离，通过接口进行数据传输（JSON格式）。
# 目的
这是给一个学姐写的毕业生设计，在这当中我学会了掌握Java websocket的注释标签式的编写方式和前端js调用系统视频和音频等信息的能力。
